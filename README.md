# Nitrate Online — Legacy Archive

**⚠️ LEGACY ARCHIVE STATUS ⚠️**

This is the **legacy archive branch** of Nitrate Online, preserving the original website design and structure from 1996–2004. This branch contains the historical content in its original format and is **not intended for public search indexing**.

---

## About Nitrate Online

An independent film criticism publication founded by Carrie Gorringe in 1996. Nitrate Online published in-depth reviews, feature essays, and international film festival coverage until 2004, operated by Nitrate Productions, Inc.

The name refers to the nitrate film stock used in cinema's early decades — a material of great beauty, sensitivity, and historical significance that could easily combust — much like the ephemeral nature of film criticism itself.

**Modern site:** [nitrateonline.com](https://nitrateonline.com)  
**Legacy archive:** [legacy.nitrateonline.com](https://legacy.nitrateonline.com) _(private archive)_

- **Search indexing blocked** via multiple prevention mechanisms

## Branch Purpose

This legacy branch serves as a **historical archive** and **design preservation** of Nitrate Online's original appearance and structure. It is maintained separately from the modern site to:

- Preserve the original aesthetic and user experience
- Maintain historical integrity of the content presentation
- Provide a reference for the site's evolution over time
- Archive content that may have been reformatted in the modern version

## Archive Access

**This branch is not meant for public discovery** and has search engine indexing disabled. Access is provided primarily for:

- Historical research and reference
- Content verification and comparison
- Design and development history documentation
- Authorized academic or journalistic use

## Modern Version

For the current, actively maintained version of Nitrate Online with modern web standards, responsive design, and updated content formatting, visit the main branch at [nitrateonline.com](https://nitrateonline.com).
